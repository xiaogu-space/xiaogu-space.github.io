Mac brew 常用命令

Apr 03, 2024
阅读时长: 1 分钟
  • 查看brew的帮助
1

brew –help
  • 安装软件
1

brew install git
  • 卸载软件
1

brew uninstall git
  • 搜索软件
1

brew search git
  • 显示已经安装软件列表
1

brew list
  • 更新软件，把所有的Formula目录更新，并且会对本机已经安装并有更新的软件用*标明。
1

brew update
  • 更新某具体软件
1

brew upgrade git
  • 显示软件内容信息
1

brew info git
  • 用浏览器打开
1

brew home
  • 显示包依赖
1

brew deps
  • 显示包的依赖树
1

brew deps --installed --tree
  • 启动web服务器，可以通过浏览器访问http://localhost:4567/ 来同网页来管理包
1

brew server
  • 删除程序，和upgrade一样，单个软件删除和所有程序老版删除。
1
2

brew cleanup git
brew cleanup
  • 查看那些已安装的程序需要更新
1

brew outdated
